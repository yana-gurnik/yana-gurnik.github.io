01 Растр та вектор

Растрова графіка
описує зображення, як масив пікселів (точок)
масштабнозалежна
має суттєвий розмір файлу
отримуються внаслідок роботи певних пристрїв (сканери, цифровіфотоапарати)
легко представляє об’єкт з великою кількістю неперервних тонів з плавними переходами між ними
Векторна графіка
описує зображення, використовуючи матиматичну модель сплайнів (вектори)
масштабнозалежна
має відносно малий розмір файлів
використовується для представлення логотипу, емблем, інфографіки, піктограм та зображень з обмеженою кольоровою палітрою
отримується внаслідок абстрації (мальована людиною)
важко представляє об’єкт з великою кількістю
неперервних тонів
плавними переходами між ними
Порада:
Визнач який вид графіки краще підійде для реалізації твого завдання та обери його

02 Кольорова модель

Це модель опису представлення кольорів
у вигляді наборів чисел.
Буває двох видів — CMYK і RGB.
CMYK
субтрактивна колірна модель
абревіатура слів блакитний, пурпуровий, жовтий, чорний
обмежене колірне охоплення
пов'язана з роботою друкарського обладнання
RGB
адаптивна колірна модель
абревіатура слів червоний, зелений, синій
необмежне колірне охоплення
пов'язвана робота моніторів, сканерів та цифровихфотоапаратів
Порада:
Для отримання найбільшпрогнозованої роздрукувки
варто обрати субтрактивну колірну модель CMYK та корстуватися пантонами

03 DPI та PPI

DPI та PPI («Dots Per Inch» та «Pixels Per Inch»)
терміни, якими позначають роздільну здатность зображення.
DPI
крапки на дюйм
роздільна здатність, додрукарського та друкарського
обладнання (друкарські верстати)
PPI
пікселі на дюйм
роздільна здатність пристроїв відображення (монітор, сканер, цифровийфотоапарат)
роздільна здатність зображень, від якої залежить кінцевий розмір зображення в друці
Порада:
Щоб отримати чітке зображення обирай PPI — 300
пікселей на дюйм і шукай друкарню з якісним
обладнанням

04 Кольоровий профIль

Кольоровий профіль — це файл, що описує охоплення кольорів в пристроях відображання та друку.
Вони діляться на дві групи – кольорові простори CMYK та кольорові простори RGB.
Кольорові простори CMYK:
допомагають передбачувано працювати з кольором
описують кольорове охоплення друкарського обладнання
можуть бути стандартом певної технології друку(для стандартів друку)
Кольорові простори RGB:
допомагають передбачувано працювати з кольором
описують кольорове охоплення для пристроїв відображення і отримання зображення(сканерів, фотоапаратів та моніторів)
Порада:
Щоб отримати потрібні кольори при друці обирай CMYK

05 Формат файлу

Формат файлу — це усталений стандарт запису інформації у файлі даного типу.
Платформонезалежні векторні формати:
AI (Adobe Illustrator) – використовується для роботи в Adobe.
EPS (Encapsulated PostScript) — використовується переважно для друку
VG (Scalable Vector Graphics)– використовується в інженерній графіці та під час розробки веб-сайтів.
CDR (CorelDraw )— файли програми CorelDraw.
Платформонезалежні растрові формати:
TIFF (Tag Image File Format) – використовується для повноколірної графіки, для друку з компресією без втрат.
JPEG, JPG (Joint Photographic Experts Group )— використовується для повноколірної графіки, для вебу. Компресія з втратами.
PNG (Portable Network Graphics) – використовується для обмежаної кольорової палітри, стиснення без втрати даних.
GIF (Graphics Interchange Format)– використовується для обмежаної кольорової палітри,створений для мережевого поширення, файли даного формату дуже малі.
Порада:
Щоб отримати потрібні кольори при друці обирай CMYK
